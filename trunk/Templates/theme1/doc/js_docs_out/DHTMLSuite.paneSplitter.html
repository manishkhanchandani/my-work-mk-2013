void loadContent(id,url,refreshAfterSeconds,onCompleteJsCode)
    This method loads content from server and inserts it into the pane with the given id If you want the content to be displayed directly, remember to call the showContent method too.

      Integer - refreshAfterSeconds - Reload url after number of seconds. 0 = no refresh ( also default)
      String - onCompleteJsCode - Js code to evaluate when content has been successfully loaded(Callback) - DHTMLSuite.paneSplitter.prototype.example = "myFunction()";. This string will be avaluated.
