Code used in this demo:

1: Create a custom function refered to by links on the page:

<script type="text/javascript">

function displayImage(imagePath,title,description)
{
	var enlargerObj = new DHTMLSuite.imageEnlarger({ isDragable:true,isModal:false });
	enlargerObj.displayImage(imagePath,title,description);
	
}

</script>

2: Create links from <A> tags:

<a href="#" onclick="displayImage('demo-images/imageEnlarger3.jpg','Title','Description.');return false">Link
