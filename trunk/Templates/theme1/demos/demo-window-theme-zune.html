The Javascript code for a window:

<script type="text/javascript">

var windowModel = new DHTMLSuite.windowModel();
windowModel.createWindowModelFromMarkUp('myWindow');

var myWindow = new DHTMLSuite.windowWidget();
myWindow.addWindowModel(windowModel);
myWindow.init();
myWindow.setStatusBarText('This is some text');

</script>

About

This is a demo of the DHTMLSuite.tableWidget class. To run this demo, you'll need to include dhtml-suite-for-applications.js or the files

  • dhtmlSuite-windowWidget.js
  • dhtmlSuite-resize.js - for the resize feature
  • dhtmlSuite-dynamiContent.js - for ajax content
  • dhtmlSuite-dragDropSimple - for the drag'n drop feature

A window can be created by pure javascript or by refering to HTML markup on your page.
