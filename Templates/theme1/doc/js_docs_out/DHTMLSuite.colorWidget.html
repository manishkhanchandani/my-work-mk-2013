setCallbackOnChangeRgb

void setCallbackOnChangeRgb(functionName)
    Specify call back function which is executed when the rgb value is modified. Only the name of the function should be sent to this method. When the callback is executed, an associative array will be sent as argument. Keys in this array will be rgb,hue,saturation and brightness.
    Parameters:
      String - functionName - Name of function to execute when the rgb color is changed.
